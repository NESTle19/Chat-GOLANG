# WebSocket Chat Server

Простой чат-сервер на Go с использованием WebSocket для обмена сообщениями в реальном времени.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
go mod init chat-server
go get github.com/gorilla/websocket
go get github.com/google/uuid
```

## 📝 Архитектура
### Компоненты
Описание
ClientManager	Управляет всеми подключениями, регистрацией и трансляцией
Client	Представляет одного клиента с его соединением и каналами
Message	Структура сообщения для JSON-сериализации
### Потоки данных
Клиент подключается → wsPage() создает Client

Клиент регистрируется → manager.register

Клиент отправляет сообщение → read() → manager.broadcast

Сервер транслирует → start() → всем клиентам

Клиент получает → write() → отправка через WebSocket
