# Chat-GOLANG
WebSocket Chat Server
Описание
Это простое приложение чата на языке Go, использующее WebSocket для обмена сообщениями в реальном времени между клиентами. Сервер поддерживает множество одновременных подключений и транслирует сообщения всем активным клиентам.

Технологии
Go - язык программирования

Gorilla WebSocket - библиотека для WebSocket соединений

Google UUID - генерация уникальных идентификаторов

Требования
Go 1.11+

Менеджер зависимостей Go Modules

Установка зависимостей
bash
go mod init chat-server
go get github.com/gorilla/websocket
go get github.com/google/uuid
Запуск
Клонируйте репозиторий:

bash
git clone <repository-url>
cd <project-directory>
Запустите сервер:

bash
go run main.go
Сервер запустится на порту 12345 и будет ожидать WebSocket подключения на эндпоинте /ws.

Подключение
Для подключения к чату используйте WebSocket соединение:

ws://localhost:12345/ws
Структура приложения
Основные компоненты
Clientmanager - управляет всеми подключенными клиентами, обрабатывает регистрацию/дерегистрацию и широковещательную рассылку

Client - представляет отдельного клиента с уникальным ID, WebSocket соединением и каналом для отправки сообщений

Message - структура сообщения с полями отправителя, получателя и содержимого

Основные функции
start() - главный цикл менеджера клиентов, обрабатывающий регистрацию, удаление и трансляцию сообщений

send() - отправляет сообщения всем клиентам, кроме указанного (игнорируемого)

read() - читает входящие сообщения от клиента и транслирует их в общий канал

write() - отправляет сообщения клиенту из его личного канала

wsPage() - обрабатывает WebSocket подключения
